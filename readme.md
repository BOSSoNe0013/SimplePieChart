## Requirements

    Android SDK rev 14 (ICS)

## Usage

### Gradle dependency

    repositories{
        maven{
            url 'https://github.com/BOSSoNe0013/SimplePieChart/raw/master/releases/';
        }
    }
    dependencies{
        compile 'com.b1project.SimplePieChart:simplepiechart:1.1.0'
    }
